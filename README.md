# mb-casino# 🎰 CASINO ON MAIN
### A MATTBEAR Interactive Project

Neon lights. Midwest nights.  
Small-town chaos wrapped in arcade glow.

CASINO ON MAIN is a browser-based experimental casino and retrocade experience built by MATTBEAR — blending arcade aesthetics, slot mechanics, surreal humor, music culture, and interactive web design into one evolving digital playground.
